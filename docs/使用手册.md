## 1. 快速入门

### 方式一：运行 .exe 程序（推荐）

→ 下载 `GeoChemExtractor.exe`，双击运行即可。无需安装 Python 环境。

### 方式二：从源码运行

```bash
cd GeoChemExtractor
source venv/Scripts/activate
export PYTHONPATH=".:geochem_extractor"
python geochem_extractor/app.py
```

### 首次使用步骤

1. 点击 **文件 → 新建项目**，保存一个新 `.gce` 文件
2. 点击 **导入Excel数据**，选择已有的 `松潘甘孜花岗岩统计.xlsx`
3. 数据加载后即可在网格中浏览、筛选和分类

## 3. 数据导入

### 从 Excel 导入

1. 确保已有打开的项目（`.gce` 文件）
2. **文件 → 导入Excel数据** 或点击工具栏 📥 按钮
3. 选择 `松潘甘孜花岗岩统计.xlsx`
4. 软件自动识别 Sheet，默认导入 "SPGZ ALL" 表
5. 等待进度条完成，数据即在网格中显示

### 数据列说明

导入数据需包含以下 94 列（软件会自动映射）：

| 分类 | 列数 | 示例列 |
|------|------|--------|
| 身份信息 | 6 | 矿区、岩体、Sample.No、Granite type |
| 年代学 | 6 | 成岩年龄、error、Method |
| 空间坐标 | 3 | X (经度)、Y (纬度)、Hight |
| 主量元素 | 12 | SiO2、TiO2、Al2O3、MgO、CaO、K2O... |
| 微量元素 | 25 | Li、Rb、Sr、Zr、Nb、Ba... |
| 稀土元素 | 14 | La、Ce、Nd、Sm、Eu、Gd、Yb、Lu |
| 同位素 | 14 | 87Sr/86Sr、143Nd/144Nd、εNd(t)、176Hf/177Hf、εHf(t) |
| 计算指数 | 4 | ASI、Ga/Al×10⁴、FeOt/(FeOt+MgO)、TZr(°C) |

## 4. PDF 表格提取

### 操作步骤

1. 点击 **导入PDF文献** 或工具栏 📄 图标
2. 选择导入方式：
   - **选择单个PDF**：处理一篇文献
   - **选择文件夹**：批量递归扫描所有 PDF
3. 软件自动：
   - 检测表格所在页面
   - 提取表格内容（文字 + 图片 OCR）
   - 显示处理进度
4. 完成后在 **表格预览** 标签页查看提取结果

### 提取策略

| 方法 | 适用场景 |
|------|----------|
| **pdfplumber** | 文字型 PDF，不规则表格（主力） |
| **PyMuPDF** | 带框线的标准表格 |
| **启发式解析** | 无框线纯文本表格 |
| **EasyOCR** | 扫描版/图片嵌入表格（自动启用） |

### 当前限制

- 中文学术 PDF 的化学式可能存在下标丢失（SiO₂ → SiO），列识别引擎已对此做了优化
- 纯图片型 PDF 的表格提取准确度依赖 OCR 质量（复杂排版表格提取率约 60-80%）

## 5. 数据管理

### 筛选数据

在左侧 **筛选面板** 中可以：

- 下拉选择矿区
- 模糊搜索岩体名称
- 筛选花岗岩类型（I/S/A/M/未分类）
- 设置 SiO₂ 范围（如 70-80%）
- 设置成岩年龄范围（如 200-250 Ma）
- 输入样品号模糊匹配
- 仅显示已分类/验证通过的样本

> 筛选是**实时**的，清除筛选可恢复全部数据。

### 编辑数据

- **双击**数据网格中的任意单元格可编辑
- 修改后自动标记为"已修改"
- 点击 **保存项目** 将更改持久化

### 合并冲突

当从不同 PDF 导入的样本与已有数据库中样本冲突时：

1. 软件弹出 **合并冲突解决对话框**
2. 逐字段对比已有值和新值
3. 选择策略：保留已有 / 使用新值 / 逐字段自定义
4. 可"应用到全部"批量处理

## 6. 花岗岩自动分类

### 分类方法

软件采用多步骤地球化学判定流程：

| 步骤 | 判别条件 | 判定结果 |
|------|----------|----------|
| 1 | Ga/Al×10⁴ > 2.6 且 Zr > 250ppm 且 FeMgI > 0.85 | **A型**（高温无水碱性） |
| 2 | K₂O < 0.8% 且 Na₂O > K₂O 且 SiO₂ > 63% | **M型**（岛弧玄武质分异） |
| 3 | ASI > 1.1（过铝质） | **S型**（沉积岩源区） |
| 4 | ASI < 1.0（准铝质） | **I型**（火成岩源区） |
| 5 | ASI 1.0-1.1（过渡区） | 综合 Na₂O/FeMgI 判定 |

### 亚型识别

- **S型亚型**：SG（高FeMgI > 0.8） / SC（低FeMgI）
- **A型亚型**：A1（Nb > 2Y，OIB 源） / A2（陆壳源）
- **I型亚型**：HFI（高分异I型，SiO₂>70%, Al₂O₃<14%）

### 操作步骤

1. 加载数据后点击 **自动分类** 按钮
2. 软件弹出分类结果对话框，展示每样本的：
   - 自动分类结果及置信度（高/中/低）
   - 关键计算指数（ASI, Ga/Al, FeMgI, TZr）
3. 可**手动覆盖**：选中行 → 下拉修改 → 应用覆盖
4. 点击 **确认并应用分类** → 数据网格行按分类着色

### 置信度说明

| 标识 | 含义 |
|------|------|
| **高** (绿色) | 多个指标一致指向同一类型 |
| **中** (橙色) | 至少一个主要指标满足条件 |
| **低** (红色) | 数据不足或落入过渡区，建议人工复核 |

## 7. 数据导出

### Excel 导出（推荐）

1. 点击 **导出数据** 或 Ctrl+E
2. 选择格式：**.xlsx**（推荐，支持分组 + 着色）
3. 选择分组方式：
   - **不分组**：全部数据一个 Sheet
   - **按花岗岩类型**：I/S/A/M 型分 Sheet
   - **按矿区**：按矿区名分 Sheet
4. 可选包含计算指数列
5. 可选同步导出图解数据
6. 点击 **开始导出**

> 导出的 Excel 自动使用分类着色：I型=浅蓝 / S型=浅红 / A型=浅绿 / M型=浅紫

### CSV 导出

- 选择 **CSV (.csv)** 格式
- UTF-8 BOM 编码，双击即可在 Excel 中打开（中文不乱码）

### 图解数据导出

勾选"同时导出图解数据"后会额外生成：

| 文件 | 内容 |
|------|------|
| `tas_diagram.csv` | SiO₂ + Na₂O+K₂O 值 + 分类标签 |
| `ree_normalized.csv` | 球粒陨石标准化后的 REE 值（Sun & McDonough 1989） |
| `discrimination.csv` | ASI、FeMgI、Ga/Al×10⁴ 等判别图数据 |

在 **图表** 标签页中，通过下拉菜单切换：

| 图表 | 说明 |

## 9. 常见问题

### Q: 应用启动后显示空白网格怎么办？

→ 检查是否已打开一个项目（`.gce` 文件）。新建项目后需要先导入数据。

### Q: PDF 导入后没有表格？

→ 部分学术论文使用纯图片排版，自动检测可能失败。检查 PDF 是否为文字型（pdfplumber 可提取）。如为扫描件，自动启用 EasyOCR。

### Q: 花岗岩分类结果不符合预期？

→ 自动分类基于地球化学经验规则，置信度为 "low" 的结果建议人工复核。可以使用**手动覆盖**功能修正。

### Q: 如何分享项目给其他人？

→ 将 `.gce` 文件和导出的 Excel 一起发送。对方打开 `.gce` 文件即可查看全部数据和分类结果。

### Q: 软件占多少磁盘空间？

→ 应用程序约 270MB（含所有依赖）。`.gce` 项目文件远小于 Excel（1000 样本约 1MB）。

## 附录 A：ASI 指数详解

**铝饱和指数 (ASI = Al₂O₃/(CaO+Na₂O+K₂O) 摩尔比)**

- **ASI < 1.0**：准铝质 → 典型 I 型花岗岩
- **1.0 < ASI < 1.1**：过渡区
- **ASI > 1.1**：过铝质 → 典型 S 型花岗岩

> 注意：高分异 I 型花岗岩也可能 ASI > 1.0，需结合全岩化学特征综合判断。
